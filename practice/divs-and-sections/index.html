Divs & Sections

Learn how to group content using generic div containers and semantic HTML sections to match your desired page layout.

What are divs and sections?

A <div> is a generic container with no built-in meaning. Use divs when you need a wrapper purely for styling or layout purposes.

A <section> is a semantic element that groups related content into a thematic section of the page. Screen readers and search engines understand its purpose better than a plain div.

When to use each

  • Use div for layout wrappers (rows, columns, spacing groups).
  • Use section when content belongs to a distinct topic or area.
  • Use header, main, and footer for page landmarks.

Basic structure pattern

<div class="layout">
  <div class="layout__header">Header</div>
  <div class="layout__body">
    <div class="layout__sidebar">Sidebar</div>
    <div class="layout__main">Main</div>
  </div>
  <div class="layout__footer">Footer</div>
</div>
