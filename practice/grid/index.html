What is CSS Grid?

CSS Grid is a two-dimensional layout system. You define rows and columns, then place items anywhere in the grid — including spanning multiple cells.

Key properties

  • grid-template-columns — define column sizes
  • grid-template-rows — define row sizes
  • gap — spacing between grid cells
  • grid-column / grid-row — span multiple cells
