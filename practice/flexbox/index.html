What is flexbox?

Flexbox is a one-dimensional layout method for arranging items in a row or a column. Set display: flex on a container to control how its children are distributed.

Key properties

  • flex-direction — row (default) or column
  • justify-content — alignment along the main axis
  • align-items — alignment along the cross axis
  • gap — spacing between items
