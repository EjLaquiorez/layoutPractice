# Coder workspace

Your personal sandbox. Edit anything here without affecting the public site or lesson modules in `practice/`.

## Quick start

1. Open [index.html](index.html) in your browser or with Live Server.
2. Edit `index.html` and `css/styles.css` as much as you want.
3. When you want a fresh attempt, copy the `experiments/starter/` folder and rename it (e.g. `experiments/header-layout/`).

## What's in here

| Path | Purpose |
|------|---------|
| `index.html` | Your main playground — change freely |
| `css/styles.css` | Styles for the playground |
| `experiments/starter/` | Blank template to duplicate for new tries |
| `experiments/` | One folder per experiment or layout idea |

## Tips

- You can link shared site tokens from `../css/main.css` if you want the same colors and spacing, or ignore it and write everything from scratch.
- Nothing in this folder is linked from the public site navigation — it is only for you.
- Delete or rename experiment folders anytime; they are not used by the lesson pages.
