# Experiments

Create a new folder here for each layout you want to try.

## How to start a new experiment

1. Copy the entire `starter/` folder.
2. Rename it (e.g. `sidebar-layout`, `hero-grid`, `nav-bar`).
3. Open the new folder's `index.html` in your browser.
4. Build your layout in that folder's `index.html` and `styles.css`.

Delete old experiments whenever you no longer need them.
